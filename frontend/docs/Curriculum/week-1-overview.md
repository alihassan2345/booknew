# Week 1: Introduction to Physical AI

Welcome to the first week of the Physical AI & Humanoid Robotics curriculum. This week covers the fundamentals of bridging the digital-physical gap in robotics applications.

## Learning Objectives

- Understand the concept of embodied intelligence
- Learn the differences between traditional AI and Physical AI
- Explore the core themes of the course
- Set up your development environment

## Content

This week introduces the foundational concepts that will be built upon throughout the course. You'll learn about:

- The evolution from digital AI to physical/embodied intelligence
- The importance of the digital-physical gap in robotics
- Overview of the four core modules
- Introduction to the tools and technologies you'll be using

## Activities

- Read introductory materials
- Set up your development environment with the required hardware specifications
- Familiarize yourself with the course structure

:::note
This curriculum follows a 13-week breakdown as specified in the original requirements, providing a progressive learning path from basic to advanced concepts.
:::
