# UI Elements & Components

This document describes the custom UI elements and components implemented in the Physical AI & Humanoid Robotics Book website.

## Homepage Features

The homepage (`/`) serves as the entry point to the book content and includes:

- **Hero Section**: Features the main title, tagline, and call-to-action buttons
- **Module Cards**: Visual cards for each of the four core modules with navigation links
- **Visual Elements**: Custom SVG graphics and icons to enhance the user experience
- **Smooth Scrolling**: Enhanced navigation experience with smooth scrolling behavior

## Navigation Improvements

- **Home Link**: Direct link to the homepage in the main navigation
- **Enhanced Footer**: Organized links to modules and resources
- **Responsive Design**: Optimized layout for mobile and desktop devices

## UI Components

### Custom Components

- **LazyImage**: For lazy loading images to improve performance
- **SmoothScroll**: For enhanced scrolling behavior (integrated globally)

### Styling Enhancements

- **Custom CSS Variables**: Extended Infima CSS framework with custom colors and properties
- **Enhanced Code Blocks**: Improved styling for code examples
- **Admonitions**: Custom styling for warnings, notes, and other callout boxes
- **Table Styling**: Enhanced appearance for comparison tables

## Responsive Design

The UI is fully responsive and adapts to different screen sizes:

- Mobile: Single column layout with touch-friendly elements
- Tablet: Optimized two-column layouts where appropriate
- Desktop: Full multi-column layouts with enhanced visual elements

## Accessibility

The UI follows accessibility best practices:

- Proper color contrast ratios
- Semantic HTML structure
- Keyboard navigation support
- Screen reader compatibility
